# sti-pano
HTML5 panorama viewer
简易的HTML5全景展示工具

使用css构造的一个6面正方体 分别贴上纹理 实现全景效果

demo: wx.sz-sti.com/panotest

兼容性
    支持以下:
        Android-original-browser 
        Android-QQbrowser-x5(微信浏览器）
        Safari
        Chrome
    不支持以下:
        IE
        Firefox
